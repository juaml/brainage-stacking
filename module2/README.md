# Module 2 – Feature Extraction

This module extracts regional brain features from CAT12-processed
structural MRI scans for brain age prediction.

## Atlases
- Schaefer-800 (cortical)
- Brainnetome (36 subcortical regions)
- Cerebellar atlas (34 regions)

Total regions: 870  
(Paper reports 873 – difference due to cerebellar atlas definition)

## Input
CSV file generated by Module 1 containing:
- Subject ID
- Age
- Path to CAT12 GMV NIfTI file

## Output
- combined_atlas.nii.gz
- regional_mean_gmv.csv
- regional_voxels.pkl
- regional_voxels_metadata.pkl

### Terminal
```bash
python feature_extractor.py merged_data.csv ./features/ --atlas_dir ./atlases/
